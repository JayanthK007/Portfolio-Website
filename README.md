## üöÄ Getting Started

### Prerequisites

- Node.js (v14 or higher)
- npm or yarn

### Installation

1. Clone the repository or navigate to the project directory:
```bash
cd "E:\Portfolio website"
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm start
```

4. Open your browser and visit:
```
http://localhost:3000
```

### Build for Production

To create an optimized production build:

```bash
npm run build
```

The build files will be in the `build` directory, ready to be deployed.

## üì¶ Deployment

This website can be deployed to various platforms:

- **Netlify**: Drag and drop the `build` folder
- **Vercel**: Connect your GitHub repository
- **GitHub Pages**: Use `gh-pages` package
- **AWS S3**: Upload build files to S3 bucket with static hosting

### Quick Deployment to Netlify

1. Build the project: `npm run build`
2. Go to [Netlify](https://www.netlify.com/)
3. Drag and drop the `build` folder

## üé® Customization

### Colors

The main color scheme uses purple gradients. To change colors, edit the CSS variables in `src/App.css`:

- Primary gradient: `#667eea` to `#764ba2`
- Secondary gradient: `#f093fb` to `#f5576c`
- Accent gradient: `#4facfe` to `#00f2fe`

### Content

Update personal information in `src/App.js`:
- Experience data
- Education details
- Skills lists
- Certifications
- Contact information
